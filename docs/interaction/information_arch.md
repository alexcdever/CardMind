# 信息架构

> **核心原则**: 清晰、扁平、快速

---

## 1. 层级结构

### 1.1 应用结构

```
应用
└── 笔记空间
    └── 卡片列表
        └── 卡片详情
```

**特点**: 最多 3 层，避免深层嵌套

### 1.2 扁平化设计

- 卡片列表 → 卡片详情（一步直达）
- 首页即可创建卡片
- 搜索随时可用

---

## 2. 导航

### 2.1 首页
- 显示所有卡片
- 支持搜索和筛选
- 底部按钮创建卡片

### 2.2 卡片详情
- 显示完整内容
- 编辑和删除操作
- 返回首页

---

## 3. 搜索和筛选

### 3.1 搜索
- 首页顶部搜索栏
- 实时搜索结果
- 匹配标题和内容

### 3.2 筛选
- 按标签筛选
- 按时间筛选
