# 用户反馈机制

> **原则**: 即时、清晰、不打扰

---

## 1. 反馈类型

### 1.1 状态反馈
系统当前状态的即时提示：
- 加载中显示加载指示器
- 同步中显示同步图标

### 1.2 结果反馈
操作完成或失败的结果提示：
- 成功：绿色提示
- 失败：红色提示
- 警告：橙色提示

### 1.3 进度反馈
长时间操作的进度显示：
- 进度条
- 百分比

### 1.4 空状态
无数据时的引导：
- 说明当前状态
- 提供操作引导

---

## 2. 同步状态

| 状态 | 显示 | 说明 |
|------|------|------|
| 同步中 | 旋转图标 | 正在同步数据 |
| 已同步 | "已同步"文字 | 同步完成 |
| 同步失败 | 橙色感叹号 | 点击查看详情 |

---

## 3. 操作反馈

### 3.1 成功提示
- 卡片创建成功
- 卡片删除成功
- 设置已保存

### 3.2 错误提示
- 网络连接失败
- 密码错误
- 数据加载失败
