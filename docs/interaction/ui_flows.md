# 用户操作流程

> **核心原则**: 快速、简洁、可靠

---

## 1. 首次使用

### 1.1 创建笔记空间（第 1 台设备）

1. 打开应用
2. 设置笔记空间密码
3. 开始使用

### 1.2 加入已有笔记空间（第 N 台设备）

1. 打开应用
2. 等待发现附近设备
3. 选择要加入的设备
4. 输入密码
5. 等待同步完成

---

## 2. 日常使用

### 2.1 创建卡片

1. 首页点击右下角 **+** 按钮
2. 输入标题和内容（支持 Markdown）
3. 点击 **完成**

**交互规格**: 详见 [卡片创建交互规格](../../openspec/specs/flutter/card_creation_spec.md) (SP-FLUT-009)

**视觉设计**: 详见 [组件库 - 按钮](../design/component_library.md#1-按钮组件) 和 [输入组件](../design/component_library.md#2-输入组件)

### 2.2 编辑卡片

1. 点击卡片进入详情
2. 点击 **编辑** 按钮
3. 修改内容
4. 自动保存

### 2.3 删除卡片

1. 卡片详情点击 **⋯** 菜单
2. 选择 **删除**
3. 确认删除

**特点**: 删除后 30 天内可恢复

---

## 3. 同步

### 3.1 自动同步

- 同一网络下的设备自动发现
- 数据变更自动同步到其他设备
- 无需手动操作

### 3.2 同步状态

- AppBar 显示同步状态图标
- 同步完成显示"已同步"
- 同步失败显示橙色提示

---

## 4. 搜索

1. 首页点击搜索图标
2. 输入关键词
3. 实时显示搜索结果

**特点**: 支持标题和内容搜索

---

## 5. 设置

### 5.1 主题
- 浅色模式
- 深色模式
- 跟随系统

### 5.2 其他设置
- 字体大小
- 导出数据

---

## 6. 错误处理

| 情况 | 处理 |
|------|------|
| 网络断开 | 自动重试，恢复后同步 |
| 加载失败 | 重试或恢复备份 |
| 密码错误 | 重新输入（限 3 次） |
