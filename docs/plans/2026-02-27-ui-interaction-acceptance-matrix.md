# UI 交互验收矩阵（2026-02-27）

| 场景 | 入口 | 研发轨断言 | 体验轨阈值 | 门禁 |
| --- | --- | --- | --- | --- |
| S1 引导分流 | 先本地使用 / 创建或加入池 | 点击入口后路由正确，且无空回调 | 从引导进入主工作区不超过 1 步 | 必过 |
| S2 卡片管理 | 新建、编辑、保存、删除、检索 | 新建到编辑可达；保存有状态反馈；失败可重试 | 新建并保存在目标步数阈值内，失败后可恢复 | 必过 |
| S3 池管理 | 创建池、扫码加入、审批、退出池 | 成功/失败路径均可断言；错误码映射有下一步动作 | 失败提示可理解，恢复路径可执行 | 必过 |
| S4 设置 | 设置项与池入口 | 设置页可访问且可到池入口 | 从设置返回主任务路径可一步完成 | 必过 |
| S5 全局同步异常 | Banner/提示动作 | 异常提示可见且可跳转处理页；本地编辑不被阻断 | 同步异常不阻断本地任务，反馈时延在阈值内 | 必过 |
