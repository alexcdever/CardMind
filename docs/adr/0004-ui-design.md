# ADR-0004: UI 设计决策

**Status**: Accepted
**Date**: 2026-01-08
**Deciders**: CardMind Team

---

## Overview

本文档记录 UI 设计系统的关键决策，包括颜色、字体、间距、布局等。

---

## 1. 颜色系统决策

### 1.1 主色选择: 青色 (Teal)

**Context**: 需要选择品牌主色，区别于竞品且适合长时间使用。

**Decision**: 采用 **青色 (Teal, #00BFA5)** 作为主色。

| 竞品 | 主色 |
|------|------|
| Notion | 紫色 |
| Obsidian | 紫/蓝色 |
| **CardMind** | **青色** |

**Reasoning**:

1. **区别于竞品**: 青色更独特，视觉识别度高
2. **品牌联想**: 蓝色（可靠）+ 绿色（成长）= 青色（知识成长）
3. **视觉舒适**: 不如红橙刺激，适合长时间阅读
4. **深色模式友好**: 青色在深色背景上对比度良好

### 1.2 语义色

| 语义 | 颜色 | 用途 |
|------|------|------|
| 成功 | 绿色 | 完成状态、正向反馈 |
| 警告 | 橙色 | 提示、注意事项 |
| 错误 | 红色 | 错误、删除、危险操作 |
| 信息 | 蓝色 | 信息提示、链接 |

## 2. 排版决策

### 2.1 字体族

**Decision**: 使用系统默认字体

| 平台 | 字体 |
|------|------|
| iOS | SF Pro |
| Android | Roboto |
| Windows | Segoe UI |
| macOS | SF Pro |
| Linux | Cantarell |

**Reasoning**:
- 无需加载额外字体，减少安装包大小
- 系统字体渲染最佳，性能好
- 符合各平台设计规范

### 2.2 字号系统

| 用途 | 字号 | 行高 |
|------|------|------|
| 标题 1 | 32px | 1.2 |
| 标题 2 | 24px | 1.3 |
| 标题 3 | 20px | 1.4 |
| 正文 | 16px | 1.5 |
| 辅助文字 | 14px | 1.43 |
| 小字 | 12px | 1.5 |

## 3. 间距决策

### 3.1 栅格系统

**Decision**: 采用 **8px 栅格系统**

**Reasoning**:

1. **Flutter 适配**: Flutter 默认 8.0 spacing
2. **Material Design 标准**: Google 推荐 8dp 基准
3. **响应式缩放**: 8 可被 2、4 整除，方便响应式调整
4. **数学美感**: 8 是 2 的三次方，整除性好

### 3.2 间距层级

| 层级 | 间距 | 用途 |
|------|------|------|
| xs | 4px | 标签内间距、图标间距 |
| sm | 8px | 元素间距、按钮内间距 |
| md | 16px | 卡片内间距、组件间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 页面边距 |
| 2xl | 48px | 大区块间距 |

---

## 4. 布局决策

### 4.1 卡片最大宽度

**Decision**: 卡片最大宽度 **800px**

**Reasoning**:

1. **阅读舒适**: 60-80 字符/行是最佳阅读宽度
2. **中文特性**: 14px 字号，800px ≈ 57 汉字/行
3. **避免过宽**: 超过 1000px 阅读需要频繁转头
4. **响应式**: 在小屏幕上自动缩小

### 4.2 断点设计

| 断点 | 宽度 | 布局 |
|------|------|------|
| xs | < 600px | 单栏 |
| sm | 600px - 839px | 单栏/双栏 |
| md | 840px - 1199px | 双栏 |
| lg | 1200px - 1599px | 三栏 |
| xl | >= 1600px | 三栏（宽距） |

---

## 5. 组件圆角

**Decision**: 采用 **中等圆角 (8px-12px)**

| 组件 | 圆角 |
|------|------|
| 按钮 | 8px |
| 卡片 | 12px |
| 输入框 | 8px |
| 对话框 | 16px |
| 底部弹窗 | 20px |

**Reasoning**:
- 8px 是 Flutter 默认圆角
- 12px 卡片圆角，视觉现代感强
- 避免过大圆角（幼稚感）或过小圆角（生硬感）

## 7. 设计原则

### 7.1 核心原则

1. **简洁专注 (Simple & Focused)**
   - 最小化视觉干扰，让内容成为焦点
   - 避免过度装饰和复杂视觉效果
   - 每个界面只做一件事

2. **快速响应 (Fast & Responsive)**
   - 清晰的视觉层级，快速定位信息
   - 即时的交互反馈
   - 支持 30 秒内完成卡片创建

3. **轻量舒适 (Light & Comfortable)**
   - 柔和的颜色，减少视觉疲劳
   - 充足的留白，呼吸感强
   - 支持深色模式，适应不同环境

4. **一致可靠 (Consistent & Reliable)**
   - 统一的设计语言
   - 可预测的交互模式
   - 清晰的状态反馈
