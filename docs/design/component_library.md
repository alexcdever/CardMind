# 组件库

> **文档类型**: 视觉设计规范（非可执行规格）
> **用途**: 定义 CardMind 的可复用 UI 组件视觉规范

---

## 📋 文档说明

本文档定义 CardMind 的**可复用组件视觉规范**，包括按钮、卡片、输入框等组件的外观设计。

**注意**: 本文档只定义视觉外观，不包含交互行为。交互规格请查看 `openspec/specs/features/`。

---
