## 快速开始

### 安装

1. **Windows**: 下载并运行 `cardmind-installer.msix`
2. **Android**: 从应用商店下载或安装 APK 文件

### 首次运行

启动应用后，你会看到一个空白的欢迎界面。点击右下角的 **+** 按钮即可创建你的第一张卡片！

## 核心功能

### 1. 创建卡片

创建卡片有两种方式：

**方法一：主页面创建**
1. 在主页面点击右下角的浮动 **+** 按钮
2. 输入卡片标题和内容
3. 点击右上角的 **保存** 图标

**方法二：快捷键（桌面版）**
- Windows: `Ctrl + N`

### 2. 编辑卡片

**Markdown 支持**

CardMind 支持完整的 Markdown 语法，让你的笔记更加丰富：

```markdown
# 一级标题
## 二级标题

**粗体文字**
*斜体文字*

- 无序列表
1. 有序列表

[链接文字](https://example.com)

> 引用文字

`代码`

\```
代码块
\```
```

**实时预览**

在编辑器中点击右上角的 **眼睛图标** 可以在编辑和预览模式之间切换。

### 3. 查看卡片

点击任意卡片即可查看详细内容：

- **完整 Markdown 渲染**: 所见即所得的排版效果
- **元数据显示**: 查看创建时间和最后修改时间
- **快速操作**: 点击顶部的编辑或删除按钮

### 4. 删除卡片

**删除单张卡片**
1. 打开卡片详情页
2. 点击右上角的删除图标（垃圾桶）
3. 确认删除操作

> ⚠️ **注意**: 删除操作目前无法撤销，请谨慎操作

### 5. 搜索和筛选

**下拉刷新**
- 在主页面向下拉动列表可以刷新卡片数据

**自动排序**
- 卡片按创建时间倒序排列（最新的在最前面）

## 使用技巧

### 主题切换

CardMind 支持浅色和深色两种主题：

1. 点击主页面右上角的 **设置图标**
2. 找到 **深色模式** 开关
3. 切换主题

你的主题偏好会自动保存，下次启动时自动应用。

### Markdown 编写技巧

**快速格式化**
- 选中文字后，可以在外部添加 `**` 来加粗，或 `*` 来斜体
- 使用 `-` 或 `1.` 开头快速创建列表

**代码片段**
```markdown
使用三个反引号创建代码块：
\```rust
fn main() {
    println!("Hello, CardMind!");
}
\```
```

**任务列表**
```markdown
- [x] 已完成的任务
- [ ] 待办事项
```

### 响应式布局

CardMind 会根据你的屏幕尺寸自动调整布局：

- **手机**: 单列列表视图
- **平板/大屏**: 双列网格视图
- **桌面**: 最多三列网格视图

### 键盘快捷键（桌面版）

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 创建新卡片 |
| `Ctrl + S` | 保存卡片 |
| `Esc` | 取消编辑/返回 |

## 常见问题

### Q: 我的数据存储在哪里？

**A**: CardMind 使用本地存储，所有数据都保存在你的设备上：

- **Windows**: `C:\Users\<用户名>\AppData\Roaming\cardmind\data\`
- **Android**: `/data/data/com.cardmind.app/files/data/`

数据包括：
- `loro/*.loro`: CRDT 数据文件（源真相）
- `cards.db`: SQLite 缓存数据库

### Q: 支持多设备同步吗？

**A**: 当前 MVP 版本（v1.0.0）暂不支持同步功能。多设备 P2P 同步将在后续版本（v2.0.0）中推出。

### Q: 可以导出我的笔记吗？

**A**: 当前版本暂不支持导出功能。该功能计划在未来版本中添加。

你可以手动备份数据文件夹来保存你的笔记。

### Q: 如何备份我的数据？

**A**: 复制整个数据文件夹即可：

1. 找到数据文件夹（见上方路径）
2. 复制整个 `cardmind` 文件夹
3. 保存到安全的位置（云盘、移动硬盘等）

**恢复数据**：将备份的文件夹复制回原位置即可。

### Q: 卡片数量有限制吗？

**A**: 理论上没有限制。性能测试表明，CardMind 可以流畅处理 1000+ 张卡片：

- 1000 张卡片加载时间: < 350ms
- 列表滚动: 流畅无卡顿
- 搜索查询: < 10ms

### Q: 支持哪些 Markdown 语法？

**A**: CardMind 使用 `flutter_markdown` 渲染引擎，支持完整的 CommonMark 标准：

- 标题（H1-H6）
- 粗体、斜体、删除线
- 列表（有序、无序、任务列表）
- 链接和图片（图片支持计划中）
- 引用块
- 代码和代码块
- 表格
- 水平分隔线

### Q: 应用崩溃或出现错误怎么办？

**A**: 请按以下步骤操作：

1. **重启应用**: 大多数临时问题可以通过重启解决
2. **检查数据完整性**: 确保数据文件夹未被损坏
3. **查看日志**: Windows 用户可以在应用目录找到日志文件
4. **报告问题**: 前往 GitHub Issues 提交 bug 报告

### Q: 应用支持哪些平台？

**A**: CardMind v1.0.0 官方支持：

- ✅ Windows 10/11 (x64)
- ✅ Android 5.0+ (ARM/ARM64)

未来计划支持：
- iOS/iPadOS
- macOS
- Linux

### Q: 应用是开源的吗？

**A**: 是的！CardMind 是一个开源项目，源代码托管在 GitHub 上。欢迎贡献代码和反馈意见。
