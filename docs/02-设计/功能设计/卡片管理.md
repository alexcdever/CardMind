# 卡片管理

## 介绍

卡片管理是CardMind应用的核心功能，负责卡片的创建、编辑、查看、删除等操作，支持多设备间的数据同步和离线使用。

## 属性

- **id**: 卡片的唯一标识符
- **title**: 卡片标题，最多20字符
- **content**: 卡片内容，最多2000字符
- **createdAt**: 创建时间，毫秒级时间戳
- **updatedAt**: 更新时间，毫秒级时间戳
- **isDeleted**: 软删除标记
- **lastModifiedBy**: 最后修改者
- **networkId**: 可选，用于绑定到特定协作网络

## 功能

### 增

#### 添加卡片
1. 点击添加按钮
2. 弹出卡片编辑器（添加模式）
3. 在编辑器中输入卡片内容（标题、内容、标签等）
4. 点击保存按钮，卡片将被添加到卡片列表中

### 删

#### 删除卡片
1. 在卡片列表中选中要删除的卡片
2. 点击删除按钮
3. 弹出二次确认对话框
4. 确认删除

### 改

#### 修改卡片
1. 在卡片列表中选中要修改的卡片
2. 点击编辑按钮
3. 弹出卡片编辑器（编辑模式）
4. 在编辑器中修改卡片内容
5. 点击保存按钮，卡片将被更新

### 查

#### 卡片列表
1. 卡片列表展示所有已创建的卡片
2. 每个卡片显示标题、内容、标签等信息
3. 卡片数量：无限制
4. 排序规则：根据更新时间排序，最新的卡片显示在顶部

#### 卡片详情
1. 点击卡片列表中的卡片
2. 弹出卡片详情弹窗
3. 显示卡片的详细内容（标题、内容、标签等）
