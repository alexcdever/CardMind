# CardMind UI布局设计

## 1. 概述

UI布局设计定义了CardMind应用的界面结构、导航系统和用户交互方式，确保应用在不同平台上都能提供一致、直观的用户体验。所有设计严格基于需求文档，只包含需求文档中明确提及的功能和概念。

## 2. 整体布局架构

### 2.1 响应式设计原则
- 适配桌面端（Windows、macOS、Linux）和移动端（Android、iOS）
- 确保在不同屏幕尺寸下都有良好的显示效果

### 2.2 核心布局结构

```
┌─────────────────────────────────────────────────────────┐
│                        内容区域                          │
│  ┌─────────────────────────────────────────────────────┐│
│  │                      卡片列表                        ││
│  └─────────────────────────────────────────────────────┘│
├─────────────────────────────────────────────────────────┤
│                        底部导航                          │
└─────────────────────────────────────────────────────────┘
```

## 3. 核心页面布局

### 3.1 主页布局
- **组成**：底部导航栏和内容区域
- **默认显示**：内容区域默认显示卡片列表
- **核心组件**：
  - 底部导航栏：包含主要功能入口
  - 卡片列表：按修改时间倒序排列
  - 悬浮添加按钮：用于创建新卡片

### 3.2 网络认证界面
- **显示时机**：应用首次启动时显示
- **核心功能**：
  - 通过协作网络ID和密码加入已存在的协作网络
  - 提供跳过选项，允许用户暂时不加入任何网络
- **布局结构**：
  - 居中布局的认证表单
  - 明确的操作按钮
  - 跳过选项链接

### 3.3 网络设置界面
- **核心功能**：
  - 显示网络连接状态和当前网络ID
  - 显示在线设备列表，包括设备ID、设备昵称、设备类型
  - 支持网络重命名操作
  - 支持退出网络操作
  - 支持设置常驻网络
- **布局结构**：
  - 网络状态概览区域
  - 设备列表区域
  - 操作按钮区域
  - 常驻网络设置区域

### 3.4 卡片相关界面

#### 3.4.1 卡片列表
- **排序规则**：按修改时间倒序排序，最新的卡片显示在最上方
- **卡片显示内容**：部分标题、部分内容、创建时间和修改时间
- **交互方式**：
  - 点击卡片查看详情
  - 左滑或长按显示操作选项（编辑、删除）

#### 3.4.2 卡片详情
- **触发方式**：点击卡片列表中的任一卡片
- **显示形式**：弹窗或新页面
- **显示内容**：完整标题、完整内容、创建时间和修改时间

#### 3.4.3 卡片编辑/添加
- **触发方式**：
  - 添加：点击悬浮按钮
  - 编辑：点击卡片操作选项
- **显示形式**：弹窗
- **核心组件**：
  - 标题输入框
  - 内容输入框
  - 保存/取消按钮

## 4. 导航设计

### 4.1 底部导航栏
- **固定位置**：应用底部
- **核心导航项**：
  - 主页/卡片列表
  - 网络设置

### 4.2 页面间导航
- **卡片列表 → 卡片详情**：点击卡片
- **卡片列表 → 卡片编辑**：点击悬浮按钮或卡片操作
- **主页 → 网络设置**：底部导航
- **网络设置 → 网络认证**：操作按钮

## 5. 交互设计

### 5.1 操作反馈
- 所有交互操作应提供明确的反馈
- 加载状态显示
- 成功/失败提示
- 确认对话框

### 5.2 手势操作
- 卡片左滑：显示操作选项
- 长按卡片：显示更多操作选项
- 下拉刷新：刷新卡片列表

### 5.3 表单交互
- 实时验证
- 输入反馈

## 6. 关键交互流程

### 6.1 卡片创建流程
1. 点击悬浮添加按钮
2. 弹出编辑窗口
3. 输入卡片标题和内容
4. 点击保存按钮
5. 卡片添加到列表

### 6.2 网络加入流程
1. 应用首次启动
2. 显示网络认证界面
3. 输入网络ID和密码
4. 点击加入按钮
5. 验证成功后进入应用主页

### 6.3 设备退出网络流程
1. 进入网络设置界面
2. 点击退出网络按钮
3. 系统检查是否有绑定卡片
4. 如有绑定卡片，显示确认提示
5. 确认后退出网络，并解除卡片绑定

### 6.4 常驻网络设置流程
1. 进入网络设置界面
2. 选择常驻网络设置
3. 选取一个或多个协作网络作为常驻网络
4. 保存设置
