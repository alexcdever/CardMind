## 遇到的问题

- **问题**: Rust 集成测试 `APP_CONFIG_CONFLICT` 错误
  - **原因**: 多个测试并行执行时竞争全局配置状态
  - **解决**: 添加 `#[serial]` 属性确保串行执行

- **问题**: Flutter 测试字符串匹配失败
  - **原因**: 代码格式化导致多行调用，与测试期望的单行格式不符
  - **解决**: 将 `frb.queryCardNotes` 调用改为单行格式
