## 遇到的问题

- `flutter pub get` 初始出现哈希异常与 TLS 错误，后通过重试恢复
- Rust 新增 `Pool` 字段（`is_dissolved`、`join_requests`）后，多个测试 fixture 需要机械迁移补齐默认值
- `join_requests` 初版只存在内存模型中，`get_pool()` 经 SQLite 读回时会丢失；后改为写入 Loro 并在读取时恢复
- `cargo fmt --check` 与直接 `rustfmt` 输出不一致，根因是 `tests/integration.rs` / `integration_exact.rs` / `unit.rs` / `unit_exact.rs` 的入口上下文差异；最终通过入口文件格式化修复
- Rust integration 全量运行时有 app config 共享状态污染，最终通过 `serial` 与测试 guard 收敛
