## 决策记录

| 时间 | 决策 | 原因 |
|------|------|------|
| 12:00 | `network_id` 保留为 FFI 内部运行时句柄，不作为业务暴露概念 | 用户约束是一实例一节点一数据池，业务层不应承担 Rust 运行时对象编排 |
| 12:20 | 邀请字符串先直接以字符串形式暴露到 owner 页面，不先做二维码 | 先满足真实组网验证最短路径，避免 UI 表现层过早扩张 |
| 12:40 | `network_id` 对应持久 runtime，而不是每次 API 调用临时建 Tokio runtime | iroh endpoint 跨 runtime 复用会触发 `Internal consistency error`，必须从根上收口生命周期 |
