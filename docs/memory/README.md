# 工作日志

> 文件命名格式：`YYYY-MM-DD.md`
> 这是第二层记忆：按天记录的工作日志

## 模板

每个工作日志文件应包含以下部分：

```markdown
# YYYY-MM-DD 工作日志

## 今日完成

- [x] 完成的事项 1
- [x] 完成的事项 2

## 进行中

- [ ] 进行中的任务 1（进度：50%）
- [ ] 进行中的任务 2（进度：30%）

## 决策记录

| 时间 | 决策 | 原因 |
|------|------|------|
| 14:30 | 选择了方案 A | 比方案 B 更简单，维护成本低 |

## 遇到的问题

- 问题描述及解决方案

## 明日计划

- [ ] 计划事项 1
- [ ] 计划事项 2

## 关联文档

- [相关 spec](link)
- [相关 plan](link)
```
