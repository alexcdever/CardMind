# 产品定义

> 面向唯一用户的个人工具。上次更新：2026-06-25

---

## 一句话

一个跨平台（Android / Windows / macOS）的纯粹 Markdown 笔记工具。

---

## 痛点

- 内置备忘录不纯粹：打开看到思维导图推荐、云服务警告
- Markdown 中英文半角符号不兼容，写不了完整语法
- 没有标签系统，回头找笔记困难

---

## 不做什么

- 不做同步（LAN / P2P / 云端均不在此阶段）
- 不做多人协同编辑
- 不做平台化扩张

---

## 交互原则

以微信聊天框为审美基准：输入流畅、无多余按钮、写完即走。

核心交互：
1. 打开编辑器 → 焦点自动落编辑区，不用多点一下
2. 首行即为标题，编辑器内无独立标题字段
3. 关闭页面 → 自动保存，无弹窗确认

---

## 当前阶段

只做本地笔记体验：编辑器 + 列表 + 标签 + 搜索。

| 用途 | 方案 |
|------|------|
| 编辑器 | appflowy_editor（WYSIWYG，原生 Markdown 支持） |
| 本地存储 | sqflite（SQLite） |
| 跨平台框架 | Flutter |
