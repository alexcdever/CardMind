# 池创建规格

## 概述

定义池创建的业务规则：名称必填、池 ID 使用 UUID v7、secretkey 明文保存于池元数据；创建成功后设备自动加入并触发同步。

---

## GIVEN-WHEN-THEN 场景

### 场景：使用有效名称与密钥创建池

- **GIVEN**: 设备未加入任何池，名称非空且提供 secretkey
- **WHEN**: 用户发起创建池请求
- **THEN**: 系统生成 UUID v7 作为池 ID
- **并且**: 系统保存 secretkey 明文到池元数据
- **并且**: 设备自动加入该池
- **并且**: 同步服务对该池立即启动

### 场景：拒绝空名称创建

- **GIVEN**: 名称为空或仅包含空白字符
- **WHEN**: 用户发起创建池请求
- **THEN**: 系统拒绝创建并返回错误 `INVALID_NAME`
