Welcome to CardMind

CardMind is a cross-platform note-taking application that allows you to:

  • Create and manage notes with rich text content
  • Sync your notes across devices
  • Work offline with local storage

API Endpoints

GET /api/cards         - List all cards
POST /api/cards        - Create a new card
DELETE /api/cards/:id  - Delete a card
POST /api/sync         - Sync cards with the server

Example Card Creation

POST /api/cards
{
    "title": "My Note",
    "content": "This is the content of my note"
}
