## Why

CardMind 需要一个统一、高效的笔记卡片 UI 组件来展示用户的笔记内容。当前应用缺乏标准化的卡片展示方式，影响用户体验和界面一致性。实现 note-card 组件将为整个应用建立基础的视觉语言，确保桌面端和移动端都能提供最佳的浏览和交互体验。

## What Changes

- **新增**: 创建 NoteCard Flutter Widget 组件，支持桌面端和移动端差异化展示
- **新增**: 实现上下文菜单系统（桌面端右键菜单 + 移动端长按菜单）
- **新增**: 添加时间格式化工具类，支持相对时间和绝对时间显示
- **新增**: 实现文本截断功能，处理长标题和内容的省略显示
- **新增**: 创建编辑器集成（桌面端模态对话框 + 移动端全屏编辑器）
- **新增**: 完整的测试覆盖（8个单元测试 + 35个Widget测试）

## Capabilities

### New Capabilities
- `note-card-display`: 笔记卡片的基础展示功能，包括标题、内容预览和时间显示
- `note-card-interaction`: 卡片交互行为，包括点击、长按、右键菜单等
- `time-formatting`: 时间格式化功能，支持相对时间和绝对时间显示
- `text-truncation`: 文本截断功能，处理单行和多行文本的省略显示

### Modified Capabilities
- 无现有能力修改，这是全新的 UI 组件功能

## Impact

**受影响的代码模块**:
- `lib/widgets/` - 新增 note-card 相关组件
- `lib/models/` - 可能需要扩展 Card 模型的 UI 辅助方法
- `lib/utils/` - 新增时间格式化和文本截断工具类
- `lib/screens/` - 新增编辑对话框和全屏编辑器

**新增依赖**:
- 可能需要引入 Flutter 手势处理和国际化相关包
- 测试相关的 flutter_test 依赖增强

**系统集成**:
- 与现有的 Card 数据模型集成
- 与未来的 P2P 同步系统兼容
- 与应用的主题系统保持一致
